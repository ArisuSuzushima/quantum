## 特性
- **快速**：使用 Go 语言编写，无需等待，即可下载。
- **简单**：简单的 优雅的用户界面，无需学习，即可使用。
- **自由**：使用 AGPL-3.0 许可开源，您可以自由使用、修改和分发。
- **免费**：永久免费，无需付费，无需广告。
- **跨平台**：支持 Windows、macOS 和 Linux。
- **多线程**：支持多线程下载，加快下载速度。
- **断点续传**：支持断点续传，无需重新下载。
